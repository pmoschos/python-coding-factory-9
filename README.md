## Getting Started 🚀

### 1. Python Installation 🐍

To run the scripts in this repository, you must have Python 3.x installed. Follow the instructions below for your operating system:

#### 🪟 Windows
You can install Python on Windows using either of the following methods:

* **Method A: via Chocolatey (Recommended for command-line users)**
  Open **PowerShell** as **Administrator** and run:
  ```powershell
  choco install python -y
  ```

* **Method B: Manual Installer**
  1. Download the latest installer from the official [Python Downloads Page](https://www.python.org/downloads/windows/).
  2. Run the installer.
  3. **CRITICAL:** Ensure you check the box **"Add python.exe to PATH"** at the bottom of the installer window before clicking **Install Now**.

#### 🍎 macOS
You can install Python on macOS using either of the following methods:

* **Method A: via Homebrew (Recommended)**
  Open your terminal and run:
  ```bash
  brew install python
  ```

* **Method B: Manual Installer**
  Download and run the macOS installer package (`.pkg`) from the official [Python Downloads Page](https://www.python.org/downloads/mac-osx/).

#### 🐧 Linux (Ubuntu / Debian)
Open your terminal and run the following command to update your packages and install Python 3 and `pip`:
```bash
sudo apt update && sudo apt install python3 python3-pip -y
```

#### 🔍 Verification
Verify that Python has been installed successfully by opening your terminal or command prompt and running:
```bash
python --version
# or
python3 --version
```

---

### 2. Clone the Repository 📂
Clone this repository to your local machine:
```bash
git clone https://github.com/pmoschos/python-coding-factory-9.git
```

### 3. Running the Scripts 💻
1. Navigate to the specific chapter folder (e.g., `chapter01`).
2. Open your terminal or command prompt in that directory.
3. Run the script of interest:
   ```bash
   python script_name.py
   # or
   python3 script_name.py
   ```
4. Each script is self-contained and heavily documented with inline comments explaining its functionality.
